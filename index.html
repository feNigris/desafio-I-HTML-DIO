DIO - Desafio I

Criando uma página web com as tags e atributos aprendidos

h1 a h6: Definem cabeçalhos, com h1 sendo o mais importante e h6 o menos importante

p: Define um parágrafo

mark: Destaca texto com fundo amarelo

small: Reduz o tamanho do texto.

i: Coloca o texto em itálico.

u: Sublinha o texto.

strong: Destaca texto em negrito, indicando ênfase.

  1. ol: Cria uma lista ordenada (numerada).
  • ul: Cria uma lista não ordenada (com marcadores).
  • li: Define um item dentro de uma lista (ol ou ul).
a: Cria um link para outra página ou recurso.

hr: Insere uma linha horizontal para separar conteúdo.

sub: Define texto como subscrito (abaixo da linha base).

ub: Define texto como sobrescrito (acima da linha base).

blockquote: Define uma citação em bloco, geralmente com recuo.

font: Define o estilo de fonte (tamanho, cor e tipo), mas é obsoleta e não recomendada no HTML moderno.

del: Indica texto que foi removido do documento, geralmente exibido com uma linha sobre ele.
